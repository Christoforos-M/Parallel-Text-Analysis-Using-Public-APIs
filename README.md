# Παράλληλη Ανάλυση Κειμένου μέσω Ανοιχτών APIs (Java)

Σε αυτό το project χρησιμοποιούνται ανοιχτά APIs που επιστρέφουν απλό κείμενο μέσω HTTP GET, από το οποίο εξάγονται στατιστικά στοιχεία με χρήση πολυνηματικής επεξεργασίας σε Java.

Ενδεικτικά APIs:

- https://loripsum.net/api/10/plaintext  
- http://metaphorpsum.com/paragraphs/10  

Το API επιλέγεται παραμετρικά κατά την εκτέλεση.

## 📚 Πληροφορίες
- Μάθημα: (συμπλήρωσε)
- Υποεργασία: Ανάλυση Κειμένου από APIs
- Γλώσσα: Java
- Φοιτητής: (το όνομά σου)

## 📝 Περιγραφή

Το πρόγραμμα δημιουργεί 1, 2, 4 ή 8 νήματα.  
Κάθε νήμα πραγματοποιεί k HTTP GET κλήσεις προς το επιλεγμένο API και συλλέγει τα παραγόμενα κείμενα.

Από το σύνολο όλων των κειμένων υπολογίζονται:

1. Ο μέσος όρος μήκους των λέξεων  
2. Το ποσοστό εμφάνισης κάθε χαρακτήρα του αγγλικού αλφαβήτου (A–Z), με άθροισμα 100%

Για όλους τους υπολογισμούς αγνοούνται τα σημεία στίξης.

Σε περίπτωση αποτυχίας κάποιας κλήσης (π.χ. πρόβλημα δικτύου), θεωρείται ότι επιστρέφεται κενό κείμενο.

Το πρόγραμμα μετρά επίσης τον συνολικό χρόνο εκτέλεσης για διαφορετικό πλήθος νημάτων.
